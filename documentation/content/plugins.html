Менеджер плагинов

Protodb обладает возможностью расширения функционала за счет плагинов. На текущем этапе развития программы, пользователь может добавлять поддержку новых типов устройств, а также скриптовых языков. Поддерка языка LUA также реализована в виде плагина.

Для управления процессом загрузки плагинов был разработан соответствующий менеджер. С его помощью пользователь сможет пометить загружаемые плагины, отключить ненужные в данный момент или сломанные, посмотреть подробную информацию о каждом из них.

Менеджер загружает плагины из двух папок: основной системной и пользовательской. В основной папке хранятся плагины, идущие вместе с самой программой, для доступа к этой папке нужны соответствующие права. Новый плагины можно добавлять путём копирования в указанную пользователем папку. Путь к этой папке можно задать через настройки приложения.

Зелёная галочка рядом с именем плагина говорит об успешной загрузке. Красный крестик наоборот, плагин небыл загружен. Если в процессе загрузки плагина произошла ошибка, строка плагина будет выделена красным.
